# サンプルコードの利用方法

```

git clone git@github.com:blackraccoon000/sample_codes.git

// 2. クローンしたリポジトリのルートディレクトリに移動
code 20260330

// 3. VSCodeのRemote Containers機能を使用して、devcontainer.jsonに基づいて開発環境を構築
pnpm install
pnpm dev
```
